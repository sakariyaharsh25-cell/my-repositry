## Image Guidelines

1. **Format**: JPG or PNG
2. **Optimization**: Compress images for web (use tools like TinyPNG)
3. **Quality**: High quality but optimized for web
4. **Aspect Ratio**: 
   - Hero/Feature images: 3:2 ratio (1200x800px)
   - Team photos: 1:1 ratio (square)
   - Portfolio: 3:2 ratio (1200x800px)
